# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.4.x   | :white_check_mark: |
| < 0.4   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT open a public issue.**
2. Email [i.pinchuk.work@gmail.com](mailto:i.pinchuk.work@gmail.com) with:
   - Description of the vulnerability
   - Steps to reproduce
   - Impact assessment
3. You will receive acknowledgment within 48 hours.
4. A fix will be developed and released as soon as practical.

We appreciate responsible disclosure and will credit reporters in the release notes (unless anonymity is requested).
